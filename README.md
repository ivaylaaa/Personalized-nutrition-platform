Full-Stack Platform for Personalized Nutrition Plans
Overview
This is a full-stack web application designed to generate automated, personalized meal plans based on user-specific data. It was developed as a graduation project for the National Professional High School of Computing Technologies and Systems (NPG po KTS).


Key Features

Automated Meal Planning Algorithm: Developed a custom algorithm that calculates and generates weekly menus based on user weight, height, age, and fitness goals.


Three-Tier Access System: Implemented a secure authentication system with three levels of access: Guest, User, and Admin.


Dynamic Admin Panel: A functional dashboard for managing recipes, macronutrients, and user content.


Relational Database: Designed a complex MySQL database structure to store and manage food data, calorie counts, and user sessions.

Tech Stack

Backend: PHP 


Frontend: HTML5, CSS3, JavaScript (dynamic UI components) 



Server/Database: Apache, MySQL 


Session Management: Secure PHP Sessions 

How it Works

Registration: Users input their physical metrics and goals.


Calculation: The system processes the data and assigns a caloric target.


Generation: A weekly plan is automatically populated from the database.


Admin Control: Admins can update the database and monitor user activity via the panel.
